# Life
 Implementation of "Conway's Game of Life" with ability to customise rules. Made with Windows Forms.
# Avariable rules presets
 - Default (B3/S23)
 - Day & Night (B3678/S34678)
 - HighLife (B36/S23)
 - Life without Death (B3/S012345678)
 - Seeds (B2/S)
